# トークべた書き支援ライブラリ「あやりりすEX」

**Source:** トークべた書き支援ライブラリ

## 概要

YAYA用ゴースト開発において、SAKURAScriptでのトークを簡潔に記述できるよう支援するライブラリ。

**利点:** ウエイトタグの記述が不要になり、複雑なスクリプト構文を削減できる。既存のSAKURAScriptとの互換性も保つため、段階的な導入が可能。

## 基本的な書き方

```
ま０５：てすとてすと
し１０：うまくいくやろか。
```

形式: キャラクター識別字（設定した1文字）＋サーフェス番号：テキスト

## 主要機能

| 機能 | 説明 |
|------|------|
| ランダム選択 | パイプ記号で区切った複数テキストから随時選択 |
| チェイントーク | 等号5つ以上で区切ると連続再生 |
| 待機時間制御 | 「待ち無し」「待ち多め」など指定可能 |
| メニュー表示 | クリック選択肢の自動生成 |
| コメント対応 | `＃` で始まる行は無視される |

**動作方式:** 字句を自動でウエイト挿入することにより、句読点での待機時間が自動調整される。

## 導入方法

1. システム辞書と YAYA の最新版を更新
2. 専用辞書ファイル2つを取得・配置
3. `yaya.txt` に読込命令を追記
4. 設定辞書でキャラ識別字を定義
