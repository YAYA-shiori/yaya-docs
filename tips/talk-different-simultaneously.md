# 同時に違うことを喋る

**Source:** Tips/同時に違うことを喋る

## 概要

`\0` と `\1` でキャラクターが同時に異なるセリフを話す方法を解説します。`\_q` と `\_w[20]` を使った擬似的な方法の代わりとなる、より効率的なソリューションが紹介されています。

## 方法

### 方法 1: SAORI を利用する

「parallel」という専用 SAORI を活用する方式です。外部ツールに依存しますが、シンプルに実装できます。

### 方法 2: スクリプトのみで実装する

添付ファイル「aya_parallel.txt」として提供されているサンプル辞書ファイルを使用します。SAORI に依存せずスクリプトだけで実現します。

### 方法 3: ぱたぱたさん（簡易的なアプローチ）

「ぱたぱたさん」（Bottle Tool 用のエディタツール）を使う方法です。SSTP Bottle 用特殊スクリプト編集ツールですが、通常のゴースト開発にも活用できます。

## 説明

通常の `\_s` タグは 0 と 1 に同じセリフを話させますが、これらの方法では異なるセリフを同時に話させることができます。

`\_q` と `\_w[20]` を組み合わせた擬似的な方法でも似た効果を得られますが、タイミングの制御が難しいという問題があります。上記の方法を使うと、より正確に同時発話を制御できます。
