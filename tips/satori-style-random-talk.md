# ランダムトークをさとりっぽく

**Source:** Tips/ランダムトークをさとりっぽく

## 概要

AYA5 辞書内で里々（さとり）風スクリプト構文を使ってランダムトークを実装する方法を説明しています。添付の `satori_for_aya.dic` を読み込むことで、里々形式のタグが AYA 辞書内で使用できるようになります。

> 「AYA辞書内で里々風スクリプトものを使うことが出来ます。しかしながら、基本的には当然AYAなので」

> 「バグ潰しなんてやってません。動かなかったら、ごめん」（作成者: ちに）

## 説明

### 対応タグ

| タグ | 説明 |
|------|------|
| `（数字）` | サーフェス番号の指定 |
| `＠` | `＠テキスト` 形式（`＞` と同一処理） |
| `＞` | `＞テキスト` 形式（`＠` と同一処理） |
| `＿` | 選択肢を生成。`（＿テキスト）` は `\\_q[テキスト,＿テキスト]` と同等 |

里々風スクリプトは `RandomTalk.Satori` 関数内でのみ有効です。

### 使い方

1. `aya_aitalk.dic` の `OnAiTalk.RandomTalk` 部分を `OnAiTalk.Satori` に置き換える（または追加する）
2. トーク内容を `RandomTalk.Satori` 関数内に里々風スクリプトで記述する

### テスト方法

「玉」などのツールで動作確認する際の SHIORI/3.0 リクエスト例:

```
GET SHIORI/3.0
ID: OnAiTalk.Satori
Sender: CallAIMist
SecurityLevel: local
Charset: Shift_JIS
```
