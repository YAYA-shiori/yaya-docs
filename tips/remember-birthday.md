# 誕生日を覚える

**Source:** Tips/誕生日を覚える

## 概要

ユーザーの誕生日をゴーストが記憶する機能の実装方法を説明します。InputBoxを使って誕生日の入力を促し、変数に保存します。

## コード例

### 誕生日入力関数（文バージョン4）

```
ChoiceBirthdayEntry {
    "\0誕生日を教えてください。\n\n\
形式はYYYYMMDDとか、ＹＹＹＹ年ＭＭ月ＤＤ日とか、\n\
YYYY/MM/DD等でお願いします。\![open,inputbox,OnInputBirthday,-1]\e"
}
```

### 入力結果の処理（文バージョン4）

Birthday.txtの内容を辞書に組み込み、`OnInputBirthday`コールバック関数で処理を実装します。

### 文バージョン5での変更点

バージョン5では`SUBSTR`関数の動作が変わり、「半角・全角問わず1文字を1としてカウントする」ようになったため、`SUBSTR`を使った文字列処理部分を修正する必要があります。

修正版サンプルとして `Birthday_revised_edition.dic`（4.3KB）が配布されていました。

## 説明

実装の概要：

1. メニューなどから誕生日入力画面を呼び出す
2. InputBoxで「YYYYMMDD」や「YYYY年MM月DD日」などの形式で入力してもらう
3. `OnInputBirthday`コールバックで入力値を受け取り、変数に保存する
4. `GETTIME`などと組み合わせて誕生日チェックを実装する

文バージョン4と5では`SUBSTR`の動作が異なるため、バージョンに応じた実装が必要です。
