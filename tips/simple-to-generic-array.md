# 簡易配列→汎用配列の変換

**Source:** Tips/簡易配列→汎用配列の変換

## 概要

カンマ区切りなどで区切られた簡易配列（デリミタ区切り文字列）を、YAYAの汎用配列（ユニバーサル配列）に変換する方法を説明します。

## 現在の方法（最新バージョン）

`SPLIT`関数を使用します。

```
i = SPLIT("Sakura,Seriko,Mayura,Naru", ",")
```

`SPLIT`関数は単純な区切り文字による分割だけでなく、さまざまな用途に使用できます。詳細はマニュアルを参照してください。

## 旧来の方法

以前のバージョンでは`RE_SPLIT`関数（正規表現ベース）が使われていました。

```
i = RE_SPLIT("Sakura,Seriko,Mayura,Naru", ",")
```

## 説明

`SPLIT`と`RE_SPLIT`の主な違い：

| 項目 | SPLIT | RE_SPLIT |
|------|-------|----------|
| 正規表現 | 不要 | 必要 |
| シンプルな分割 | 向いている | 過剰な場合がある |
| パターンマッチ | 非対応 | 対応 |

単純な区切り文字による分割には`SPLIT`を使うと簡潔に書けます。複雑なパターンによる分割が必要な場合は`RE_SPLIT`を使用します。
