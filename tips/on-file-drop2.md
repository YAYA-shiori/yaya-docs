# OnFileDrop2の使い方

**Source:** Tips/OnFileDrop2の使い方

## 概要

ゴーストにファイルをドラッグ＆ドロップすると `OnFileDrop2` イベントが発生する。このイベントを利用することで、壁紙の変更や音楽データの再生などの機能を実装できる。

## 応用例

- 壁紙変更機能
- 音楽データ再生機能

## 必要なSAORI

実装には以下の外部DLLが必要：

| SAORI | 作者 |
|-------|------|
| MCIAudioR | umeiさん |
| set_wallpaper | 櫛ヶ浜やぎさん |
| PROXY.DLL | 橋本孔明さん |

## 実装サンプル

実装に必要なサンプルコード（OnFileDrop2.txt）がページの添付ファイルとして提供されている。
