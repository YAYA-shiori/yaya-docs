# 配列の要素数を取得

**Source:** Tips/配列の要素数を取得

## 概要

YAYAで配列の要素数（サイズ）を取得する方法を説明します。

## 方法1: ARRAYSIZE関数（推奨）

最もシンプルな方法は組み込みの`ARRAYSIZE`関数を使用することです。

```
i = ("A","B","C")
j = ARRAYSIZE(i)
// j == 3
```

## 方法2: ANY関数

文5には`ANY`関数があり、配列からランダムに1要素を取り出せます（要素数取得とは用途が異なりますが、配列操作の一種です）。

## 方法3: カスタム関数（dim_enum）

ユーザー投稿のカスタム関数で、文字列内のカンマ数をカウントすることで要素数を返します。

```
dim_enum {
    _s = _argv[0]
    _count = 1
    _i = 0
    for _i < STRLEN(_s) {
        if SUBSTR(_s, _i, 1) == "," {
            _count++
        }
        _i++
    }
    _count
}
```

ユーザーに覚えてもらった言葉をランダムに取り出す際の乱数範囲指定など、動的なサイズを扱う場面で役立ちます。

## 説明

通常は`ARRAYSIZE`関数を使えば十分です。カスタム関数（dim_enum）は、カンマ区切り文字列を汎用配列に変換せずに要素数だけを知りたい場合に使えますが、`SPLIT`で変換してから`ARRAYSIZE`を使う方がシンプルです。
