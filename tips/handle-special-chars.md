# 特殊文字を扱いたい

**Source:** Tips/特殊文字を扱いたい

## 概要

Unicode で特殊なコードポイントが割り当てられている文字（™ など）をバルーン表示や内部処理で使う方法を解説します。

## コード例

### バルーンに表示するだけの場合

`\_u[コード]` 構文を使用します（16 進数または 10 進数）。

```
"\0\_u[0x2122]これは商標マーク(™)です。\e"
```

### 内部処理でも使用する場合

`CHR` 関数を使用します。

```
CHR(0x2122)
```

文字列への埋め込み:

```
// 変数展開で埋め込む
"Enbug Masters%(CHR(0x2122))"

// 文字列連結で埋め込む
'Enbug Masters' + CHR(0x2122)
```

### charset.output の設定（CHR でバルーン出力する場合）

`CHR` で生成した文字列をバルーンに正しく出力するには、基礎設定ファイルに以下を追記します。

```
charset.output, UTF-8
```

既存の `charset.output` 設定がある場合は、右辺の値を `UTF-8` に変更してください。

## 説明

### \_u[コード] と CHR の違い

| 方法 | 用途 | 注意点 |
|------|------|--------|
| `\_u[コード]` | バルーン表示のみ | 内部文字列として使えない |
| `CHR(コード)` | 内部処理・文字列操作 | バルーン出力には charset.output 設定が必要 |

### コードの指定方法

16 進数と 10 進数の両方が使えます。

```
// 16進数
CHR(0x2122)   // ™ (TRADE MARK SIGN)

// 10進数
CHR(8482)     // 同上
```

### よく使う特殊文字の例

| 文字 | Unicode | 16 進数 |
|------|---------|---------|
| ™ | U+2122 | 0x2122 |
| © | U+00A9 | 0x00A9 |
| ® | U+00AE | 0x00AE |
| ♪ | U+266A | 0x266A |
