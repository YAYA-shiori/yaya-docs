# ゴーストのインストール日時を取得する

**Source:** Tips/ゴーストのインストール日時を取得する

## 概要

ゴーストがインストールされた日時を取得する方法。最新バージョンのYAYAのみ対応。

## コード例

### インストール日時の取得

```
GETTIME(FATTRIB('../../')[9])
```

`FATTRIB('../../')[9]` はゴーストのルートディレクトリの属性情報の9番目（インストール日時のタイムスタンプ）を返す。

`GETTIME()` に渡すことで、以下の要素を含む汎用配列として取得できる：

| インデックス | 内容 |
|-------------|------|
| 0 | 年 |
| 1 | 月 |
| 2 | 日 |
| 3 | 曜日 |
| 4 | 時 |
| 5 | 分 |
| 6 | 秒 |

### インストールからの経過時間

```
GETSECCOUNT() - FATTRIB('../../')[9]
```

`GETSECCOUNT()` から `FATTRIB('../../')[9]` を引くことで、インストールからの経過秒数を算出できる。
