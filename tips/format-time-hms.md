# 秒数を○時間○分○秒形式にする

**Source:** Tips/秒数を○時間○分○秒形式にする

## 概要

秒数の整数値を「○時間○分○秒」形式に変換するYAYA関数の実装例です。

## コード例

```
sec_hms {
    _v=TOINT(_argv[0])
    _a1=_v/3600
    _a2=(_v-(_a1*3600))/60
    _a3=(_v-(_a1*3600))-(_a2*60)
    _v=(_a1,_a2,_a3)
    _v
}
```

## 説明

**シグネチャ:** `sec_hms( 秒数 )`

返り値は配列で、以下の要素を持ちます。

| インデックス | 内容 |
|------------|------|
| `[0]` | 時間 |
| `[1]` | 分 |
| `[2]` | 秒 |

注意: 時間が24を超えた場合でも日数には変換せず、そのまま時間として返します。必要に応じて追加の処理を実装してください。

## 使用例

ゴーストの起動時間（`ghostuptime`）と組み合わせた使用例：

```
ゴースト起動時間 {
    _h = sec_hms(ghostuptime)[0]
    _m = sec_hms(ghostuptime)[1]
    _s = sec_hms(ghostuptime)[2]
    "\0\s[0]起動してから%(_h)時間%(_m)分%(_s)秒経ちました。\e"
}
```
