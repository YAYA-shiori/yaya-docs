# ゴースト切替えメッセージを変化させる

**Source:** Tips/ゴースト切替えメッセージを変化させる

## 概要

特定のゴーストに切り替わった際に、前のゴーストが話した内容（セリフ）に基づいて異なる反応を返す方法を説明しています。`OnGhostChanged` イベントで `reference0`（前のゴースト名）と `reference1`（前のゴーストの最後のセリフ）を参照し、キーワード検索 `_in_` 演算子で条件分岐します。

## コード例

```
OnGhostChanged
{
    if reference0 == "ゴーストA"
    {
        if "お料理" _in_ reference1
        {
            "(文章1)"
        }
        elseif "薬" _in_ reference1
        {
            "(文章2)"
        }
        else
        {
            "(文章3)"
        }
    }
}
```

## 説明

- `reference0` : 切替え前のゴースト名
- `reference1` : 切替え前のゴーストが最後に話したセリフ
- `_in_` : 文字列が含まれるかチェックする演算子
- キーワード条件を増やしたい場合は `elseif` ブロックを追加する
- 既存の辞書に追加する場合は、先頭の `if` ではなく `elseif` を使用すること
