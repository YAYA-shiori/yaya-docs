# 繰り返し

**Source:** マニュアル/基本/繰り返し

## for ループ

```
for _i = 0 ; _i < 10 ; _i++
{
  _i + "回目\n"
}
```

**注意:** YAYA では条件式をカッコで囲まない。`for(_i = 0; _i < 10; _i++)` はエラーになる。

**よく使うループ演算子:**

| 演算子 | 意味 |
|--------|------|
| `a++` | `a = a+1` |
| `a--` | `a = a-1` |
| `a+=b` | `a = a+b` |
| `a-=b` | `a = a-b` |

## while ループ

```
_i = 0
while _i < 10
{
  _i + "回目\n"
  _i++
}
```

**無限ループ:** `while 1` または `while 1 == 1` で無限ループになる（ブール値は0以外が真）。

## break 文

ループを途中で抜けるには `break` を使う：

```
_i = 0
while 1
{
  if _i >= 10
  {
    break
  }
  _i + "回目\n"
  _i++
}
```
