# 演算子

**Source:** マニュアル/基本/演算子

## 算術演算子

| 演算子 | 説明 | 例 |
|--------|------|-----|
| `x + y` | 加算 | `1+1` → `2` |
| `x - y` | 減算 | `2-1` → `1` |
| `x * y` | 乗算 | `6*2` → `12` |
| `x / y` | 除算（余りは切り捨て） | `4/3` → `1` |
| `x % y` | 剰余 | `4%3` → `1`（偶奇判定に使用可） |

## 論理演算子

if文や for 文の条件として使用する：

| 演算子 | 説明 |
|--------|------|
| `x == y` | 等値判定 |
| `x != y` | 不等値判定 |
| `x && y` | AND演算子 |
| `x \|\| y` | OR演算子 |
| `x > y` / `x >= y` / `x < y` / `x <= y` | 大小比較 |
| `!x` | 否定演算子 |
| `y _in_ x` | xの中にyが存在する |
| `y !_in_ x` | xの中にyが存在しない |

## 文字列演算

```
_c = "さくら" + "と" + "うにゅう"
// 結果: "さくらとうにゅう"
```

## 配列操作

```
_array = (_hoge, _fuga, _gonyo)
```

カンマ演算子 `,` で複数の値を配列化する。
