# 配列

**Source:** マニュアル/基本/配列

## 概要

配列は変数をまとめて扱う仕組み。個別に `_a = 1; _b = 2` と書く代わりに、まとめて管理できる。

## 基本的な使い方

```
_a = (1,2,3,4,5)
_sum = _a[0] + _a[1] + _a[2] + _a[3] + _a[4]
```

インデックスは0始まり。

## ARRAYSIZE() による動的サイズ取得

配列の長さを動的に取得することで、サイズに依存しない柔軟なコードが書ける：

```
_sweets = ("プリン", "チョコレート", "クレープ", "甘口いちごスパ")
for _i = 0 ; _i < ARRAYSIZE(_sweets) ; _i++
{
  _sweets[_i] + "\n"
}
```

## foreach ループ（最も簡単な方法）

`foreach` を使うと、インデックスや ARRAYSIZE を意識せずに配列をイテレートできる：

```
_sweets = ("プリン", "チョコレート", "クレープ")
foreach _sweets ; _item
{
  _item + "\n"
}
```
