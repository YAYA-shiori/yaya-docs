# 変数

**Source:** マニュアル/基本/変数

## 概要

変数は数値や文字列を格納するコンテナ。一度値を代入すると以前の値は上書きされる。

**重要:** 変数の宣言は関数の中でのみ行える。

## 基本的な使い方

変数の代入は `=` 演算子を使う：

```
_a = 1
_b = 2
_c = _a + _b  // 結果: 3
```

## 文字列変数

```
_a = "紅茶"
_b = "ケーキ"
_c = _a + "と" + _b  // 結果: "紅茶とケーキ"
```

## SAKURAスクリプトとの組み合わせ

スクリプトは必ず引用符で囲む必要がある：

```
// NG: そのままだと動かない
\!\[open,inputbox,OnTest,60000\]

// OK: 引用符で囲む
"\!\[open,inputbox,OnTest,60000\]"
```

## トーク内での変数表示

変数の内容をトークテキストに表示するには `%(変数名)` の構文を使う：

```
"\0\s[0]飲み物なら%(_hoge)をどうぞ\e"
```
