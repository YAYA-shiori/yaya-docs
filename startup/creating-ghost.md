# YAYAでゴーストを作る

**Source:** StartUp/YAYAでゴーストを作る

## ステップ1: テンプレートゴーストを選ぶ

既存のテンプレートから改造することを推奨。

| テンプレート | 説明 |
|-----------|------|
| **はろーYAYAわーるど（紺野ややめ）** | スタンドアロンテンプレート。そのまま楽しめ、カスタマイズも可能 |
| **SimpleYAYAテンプレート** | より最小限の出発点を求める場合に |
| **紺野りりす** | 「強力な開発サポート処理」付き。シンプルさと充実した拡張性を兼ね備えることを目指したテンプレート |
| **デバッガ（玉）** | テンプレートではないが、辞書のエラーチェックに推奨される開発ツール |

**重要:** 紺野ややめや紺野りりすを使う場合、大規模な編集前に `yaya_string.txt` の `homeurl` 設定を無効化すること（ネットワーク更新でファイルが変更される恐れがある）。
