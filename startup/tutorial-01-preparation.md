# チュートリアル1: 準備

**Source:** チュートリアル/1.準備

## 必要なダウンロード

3つのコンポーネントが必要：

1. **SSP（最新版）**: 2013年以降の主要ベースウェア。豊富な機能、軽量動作、継続的なアップデートのためSSPを前提とするチュートリアル。

2. **yaya.dll（最新版）**: YAYA の SHIORI コア。コードリポジトリから入手。

3. **テンプレートゴースト**: 複数のオプションがある。このチュートリアルでは `SimpleYAYAテンプレート` を使用。SSP でインストール後、ゴーストのmasterディレクトリ内のバンドルされた yaya.dll を最新版に置き換える。

## 開発環境のセットアップ

### SSP開発パレット

開発機能を有効化: ゴーストを右クリック → 設定 → 全般 → 「開発者用機能を有効にする」にチェック。その後、右クリック → 機能 → 開発パレット でアクセス。

開発中に常時開いておくと便利なデバッグ機能を提供する。

### YAYAデバッグツール

デバッガ「玉」をダウンロードして任意の場所に解凍。辞書のエラーチェックに使用する。

## オプション: テキストエディタの設定

Windows メモ帳でも動作するが、YAYA固有のシンタックスハイライトが可読性を向上させる。Tipsセクションに `.dic` ファイル向けの複数のカラースキームオプションがある。

## ファイル拡張子の関連付け

以下の3つの拡張子をテキストエディタに関連付ける：
- `.dic`（辞書ファイル）
- `.log`（実行ログ）
- `.cfg`（セーブデータ）

YAYA設定済みエディタを使う場合は `.dic` をそのアプリケーションに専用で関連付けることでシンタックスハイライトが活用できる。
