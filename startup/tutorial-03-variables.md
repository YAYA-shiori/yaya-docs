# チュートリアル3: 変数を使ってみよう

**Source:** チュートリアル/3.変数を使ってみよう

## 変数とは

変数は文字列や数値を格納するコンテナ。ゴースト開発の例：ユーザー名、起動回数、好感度など。

## 文字列変数と数値変数

```
// 数値代入（クォートなし）
変数 = 1

// 文字列代入（クォートあり）
変数 = "1"
```

クォートの有無で YAYA は文字列か変数参照かを判断する。

**型変換関数:**

| 関数 | 説明 |
|------|------|
| `TOSTR(変数)` | 文字列に変換 |
| `TOINT(変数)` | 整数に変換 |
| `TOAUTO(変数)` | 自動変換 |

## グローバル変数とローカル変数

| 種類 | プレフィックス | ライフタイム |
|------|--------------|-------------|
| ローカル変数 | `_変数名` | 一時的。スコープブロックを抜けると自動削除 |
| グローバル変数 | プレフィックスなし | 永続。明示的に削除するまで保持 |

**ベストプラクティス:** ユーザー名や好感度などの永続データ以外は、ローカル変数を使うこと。

## 実用例

### Reference変数の使用

```
OnMouseDoubleClick {
  "\0\s[0]"
  reference[3]
  "番のキャラがつつかれたよ！\e"
}
```

または インライン構文で：

```
"\0\s[0]%(reference[3])番のキャラがつつかれたよ！\e"
```

### カスタム変数の作成

```
OnMouseDoubleClick {
  プレゼント = "花束"
  "\0\s[0]わーい「%(プレゼント)」ありがとう！\e"
}
```

### 算術演算

```
好感度 = 好感度 + 1
好感度++
好感度 += 2
```

文字列の連結も同様：

```
プレゼント = "花" + "束"
```

## 変数の永続化

ゴーストを閉じると `yaya_variable.cfg` セーブファイルに変数が保存される：
- 文字列変数: `プレゼント,"花束",,`
- 数値変数: `好感度,1,,`
