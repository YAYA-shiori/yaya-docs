# SETTAMAHWND
**Category:** デバッグ
**Source:** マニュアル/関数/SETTAMAHWND

## Signature
```
SETTAMAHWND( hwnd )
```

## Parameters
| Parameter | Description |
|-----------|-------------|
| hwnd | デバッグツール「玉」のウィンドウハンドル（数値型） |

## Returns
なし（VOID）

## Description
デバッグツール「玉」のウィンドウハンドルを設定します。設定後はデバッグメッセージがそのウィンドウに出力されます。

## Compatibility
- YAYA: Tc562-1以降
