# LOG10
**Category:** 数学関数
**Source:** マニュアル/関数/LOG10

## Signature
```
LOG10(val)
```

## Parameters
| Parameter | Description |
|-----------|-------------|
| val | 常用対数を求める実数値 |

## Returns
- 成功: valの常用対数（底10、実数）
- 失敗: 0.0

## Description
常用対数（底10）を返します。

- AYA: バージョン5.8以降で使用可能
