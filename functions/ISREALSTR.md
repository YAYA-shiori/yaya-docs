# ISREALSTR
**Category:** 文字列操作
**Source:** マニュアル/関数/ISREALSTR

## Signature
```
ISREALSTR(string)
```

## Parameters
| Parameter | Description |
|-----------|-------------|
| string | 判定対象の文字列 |

## Returns
- 1: 実数として解釈可能
- 0: 実数として解釈不可能

## Description
与えられた文字列が実数として解釈可能かどうかを判定します。

## Compatibility
- YAYA: 初期バージョンより使用可能
- AYA: バージョン5.8以降で使用可能
