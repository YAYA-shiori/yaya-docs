# ISINTSTR
**Category:** 文字列操作
**Source:** マニュアル/関数/ISINTSTR

## Signature
```
ISINTSTR( string )
```

## Parameters
| Parameter | Description |
|-----------|-------------|
| string | 整数として解釈できるか検証する文字列 |

## Returns
- 1: 文字列を整数として解釈できる
- 0: 文字列を整数として解釈できない

## Description
指定した文字列が整数として解釈できるかどうかを検証する。ISREALSTRと対になる関数で、整数と浮動小数点数の文字列表現を区別するのに役立つ。

## Compatibility
- YAYA: 初期から利用可能
- AYA: 5.8以降
