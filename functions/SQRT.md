# SQRT
**Category:** 数学関数
**Source:** マニュアル/関数/SQRT

## Signature
```
SQRT(val)
```

## Parameters
| Parameter | Description |
|-----------|-------------|
| val | 平方根を求める実数値 |

## Returns
- 成功時: val の平方根（実数）
- 失敗時: 0.0

## Description
指定した値の平方根を返します。

## Compatibility
- YAYA: 初期バージョンより利用可能
- AYA: 5.8以降
