# ANY

**Category:** 文字列操作
**Source:** マニュアル/関数/ANY

## Signature

```
ANY(var)
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| var | ランダムに選択する対象。文字列・汎用配列・整数・実数のいずれか |

## Returns

成功時：ランダムに選ばれた要素のひとつ
失敗時：VOID

文字列を単純配列として扱う場合はその中からランダムに1要素を返す。汎用配列の場合もランダムに1要素を返す。数値型（整数・実数）の場合はその値をそのまま返す。選ばれた要素のインデックスは LSO 関数で取得できる。

## Example

```
_array = ('さくら','まゆら','ねここ')
ANY(_array)  // 「まゆら」などランダムな要素を返す

_str = 'さくら,まゆら,ねここ'
ANY(_str)    // 「ねここ」などランダムな要素を返す

LSO          // 2（「ねここ」のインデックス）を返す
```
