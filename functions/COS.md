# COS

**Category:** 数学関数
**Source:** マニュアル/関数/COS

## Signature

```
COS(rad)
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| rad | ラジアン単位の角度 |

## Returns

成功時：コサイン値を表す実数
失敗時：0.0

## Example

```
_rad = 60 * 3.14159265358 / 180  // 60° をラジアンに変換
COS(_rad)  // 0.500000 を返す
```

## Compatibility

- YAYA: 初版から利用可能
- AYA: バージョン5.8以降
