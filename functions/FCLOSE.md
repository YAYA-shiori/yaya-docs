# FCLOSE

**Category:** ファイル操作
**Source:** マニュアル/関数/FCLOSE

## Signature

```
FCLOSE( path )
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| path | FOPEN で指定した yaya.dll からの相対ファイルパス |

## Returns

返り値はありません（VOID）。

## Description

FOPEN で開いたファイルを閉じます。

## Compatibility

- YAYA: 初版より対応
- AYA: 5.8以降
