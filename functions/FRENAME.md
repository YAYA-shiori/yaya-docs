# FRENAME

**Category:** ファイル操作
**Source:** マニュアル/関数/FRENAME

## Signature

```
FRENAME( from , to )
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| from | 変更前のファイル名。フルパス指定可。相対パスはDLLのロード位置を基準とする |
| to | 変更後のファイル名 |

## Returns

- 成功時: 1
- 失敗時: 0

## Compatibility

- YAYA: 初回リリースから利用可能
- AYA: バージョン 5.8 以降
