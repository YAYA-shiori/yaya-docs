# LSO
**Category:** メタ操作・特殊関数
**Source:** マニュアル/関数/LSO

## Signature
```
LSO
LSO()
```

## Parameters
なし

## Returns
直前のセレクション操作で選択された位置を示す数値。

## Description
"last Selection Order" の略。直前に実行されたセレクション操作の選択位置を数値で返します。

LSO は選択操作の直後に呼び出す必要があります。コード内の別の場所に置くと、意図しない別のセレクション結果を取得してしまう可能性があるため注意が必要です。

出力ファイナライザが存在する場合、LSO はすべての組み合わせ結果にわたって動作します。例えばネストしたセレクションは複数の出力（0〜5の範囲など）を生成し、LSO の値は各組み合わせに対応します。

## Compatibility

- AYA: バージョン5.8以降で使用可能
