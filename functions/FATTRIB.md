# FATTRIB

**Category:** ファイル操作
**Source:** マニュアル/関数/FATTRIB

## Signature

```
FATTRIB( path )
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| path | 情報を取得するファイルまたはディレクトリのパス。フルパス・相対パスともに使用可能。 |

## Returns

11要素の汎用配列を返す。失敗した場合は `-1` を返す。

| インデックス | 内容 |
|-------------|------|
| [0] | アーカイブ属性フラグ |
| [1] | 圧縮属性フラグ |
| [2] | ディレクトリ属性フラグ |
| [3] | 隠しファイル属性フラグ |
| [4] | 通常ファイル属性フラグ |
| [5] | オフライン属性フラグ |
| [6] | 読み取り専用属性フラグ |
| [7] | システム属性フラグ |
| [8] | 一時ファイル属性フラグ |
| [9] | 作成日時（iノード更新日時）エポック秒（1970/1/1基準） |
| [10] | 最終更新日時エポック秒（1970/1/1基準） |

## Example

```
_attrib = FATTRIB('hoge.txt')
_attrib[10]          // 最終更新日時（エポック秒）
GETTIME(_attrib[10]) // 読みやすい形式に変換
```

## Compatibility

- YAYA: 初版より対応
- AYA: 5.8以降（ただし5.8より前のバージョンでは [9]〜[10] は使用不可）
