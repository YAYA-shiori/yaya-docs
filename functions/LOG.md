# LOG
**Category:** 数学関数
**Source:** マニュアル/関数/LOG

## Signature
```
LOG(val)
```

## Parameters
| Parameter | Description |
|-----------|-------------|
| val | 自然対数を求める実数値 |

## Returns
- 成功: valの自然対数（実数）
- 失敗: 0.0

## Description
自然対数（底e）を返します。

## Compatibility
- YAYA: 初期バージョンより使用可能
- AYA: バージョン5.8以降で使用可能
