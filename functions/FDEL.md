# FDEL

**Category:** ファイル操作
**Source:** マニュアル/関数/FDEL

## Signature

```
FDEL( path )
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| path | 削除するファイルのパス。フルパスまたは相対パス（DLLロード位置基準）。 |

## Returns

- `1`: 成功
- `0`: 失敗

## Compatibility

- YAYA: 初版より対応
- AYA: 5.8以降
