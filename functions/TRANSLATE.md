# TRANSLATE
**Category:** 文字列操作
**Source:** マニュアル/関数/TRANSLATE

## Signature
```
TRANSLATE( text, format_from, format_to )
```

## Parameters
| Parameter | Description |
|-----------|-------------|
| text | 置換対象の文字列 |
| format_from | 置換前の文字リスト（書式あり、後述） |
| format_to | 置換後の文字リスト。空文字列を指定すると削除 |

## Returns
- 成功: 置換後の文字列
- 失敗: -1

## Description
文字列を文字単位で高速に置換する。Perlのtr///（オプションなし）相当の機能を提供する。

### 書式

**単純な列挙**: 文字を1対1でマッピングする。リストにない文字は変更されない。
```
TRANSLATE('ぬるぽガッ', 'ぽガ', 'りゴ') // 'ぬるりゴッ'
```

**範囲指定**: 「-」を使ってUnicodeの範囲を指定して一括置換する。
```
TRANSLATE('123456789', '1-9', '①-⑨') // '①②③④⑤⑥⑦⑧⑨'
```

**エスケープ**: `\\-` でリテラルのハイフン、`\\\\` でリテラルのバックスラッシュを指定する。
```
TRANSLATE('\\-\\-\\-', '\\\\', '\\-') // '------'
```

**削除**: 第3引数を空文字列にすると、該当文字を削除する。
```
TRANSLATE('46cm三連装砲', '0-9a-z', '') // '三連装砲'
```

## Example
ひらがな・カタカナの相互変換:
```
_wokyu = TRANSLATE('ティータイムは大事にしないとネー', 'ぁ-んァ-ン', 'ァ-ンぁ-ん')
```
