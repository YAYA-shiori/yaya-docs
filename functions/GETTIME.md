# GETTIME
**Category:** システム情報
**Source:** マニュアル/関数/GETTIME

## Signature
```
GETTIME( [ sec ] )
```

## Parameters
| Parameter | Description |
|-----------|-------------|
| sec | （省略可）EPOCH（1970年1月1日 00:00:00 UTC）からの経過秒数（整数）。省略時は現在のシステム時刻を返す |

## Returns
以下の構造を持つ汎用配列:

| インデックス | 値 | 詳細 |
|------------|---|------|
| 0 | 年 | 西暦 |
| 1 | 月 | 月 |
| 2 | 日 | 日 |
| 3 | 曜日 | 0〜6（0=日曜日） |
| 4 | 時 | 24時間表記 |
| 5 | 分 | 分 |
| 6 | 秒 | 秒 |
| 7 | 年内経過日数 | 年始からの経過日数 |
| 8 | 夏時間フラグ | 0=標準時 / 1=夏時間 |

## Description
現在の日付・時刻情報を取得する。C言語の `localtime` 関数と同等の動作をする。

`sec` を指定した場合はその秒数をEPOCHからの経過時間として解釈し、対応する日時情報を返す。GETSECCOUNTと組み合わせることで、例えば10日後の日付を計算するといった用途に使える。

## Compatibility
- YAYA: 初期から利用可能
  - Tc542-1以降: 年内経過日数（インデックス7）と夏時間フラグ（インデックス8）が追加
- AYA 5.8: 引数なしのみサポート
